Projects


  • Intelligent Autonomous Car Robot

    Designed autonomous vehicle prototype that moved automatically using GPS with an Android app for the UI for entering source and destinations. Bluetooth connection between prototype and app via Arduino, lane detection using Hogs’ Algorithm and obstacle detection using ultraviolet and infrared sensors. Currently working on deep reinforcement learning for improved environment perception.


  • Dark Web Page Prediction

    Categorized dark web pages using HTML DOM structure. Trained a Graph Neural Network for DOM graphs and achieved 97% accuracy. Prepared a dataset by extracting DOM graph properties like number of nodes and used classification and clustering for models. Theoretical study of graph properties and their implication for deciding upon the web page structure.


  • Detecting Malware and Phishing Websites using AutoML

    Trained models using Brazilian malware dataset and custom dataset for phishing website, prepared by scraping URLs and feature extraction. Used TPOT library to select best-fit pipeline and Google Cloud AutoML to train state-of-the-art model on the datasets. Achieved 98% accuracy with both TPOT and AutoML for the malware dataset Achieved 91% accuracy with TPOT and 98% accuracy with AutoML for the phishing website dataset.


  • Cough Based Disease Prediction System

    Trained models for predicting the disease a patient had based on cough sound and body features like BMI, height, etc. Collected cough samples from clinic and processed them to extract audio features, and achieved 61% accuracy on models, beating the 58% state-of-the-art.


  • Face Recognition Based Greeting Robot

    Developed a robot for recognizing and automatically greeting visitors to the laboratory, “Welcome XXX”. Used 3D face recognition. Deployed in the Blockchain and AI lab of VJTI, under guidance of Dr. Dhiren Patel, Director.


  • Increasing the Session Time for Cowrie Honeypot

    Modified Cowrie honeypot using Q-learning making it adaptive to increase its interaction with the attacker and avoid detection. The sessions logs were passed to a severity analyzer module for further insights. Beat the state-of-the-art analytics results with article (10.1007/s12652-021-03229-2) Deployed in real-time in the Center of Excellence lab of VJTI, under guidance of Dr. Faruk Kazi, Dean R&D.


  • Marksheet Generator System

    Designed a Spring-Java based automated marksheet generator that fetched students’ subject wise marks from MySQL database and printed their marksheet.
